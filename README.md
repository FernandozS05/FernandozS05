## ⚡ My Projects

### 🖥️ **Website for the Campaign: #NoTeLaJueguesConLasDrogas**  
📅 **August 2023 - February 2024**  
🔹 Contributed to the development of the website for the **#NoTeLaJueguesConLasDrogas** campaign, led by the Veracruz Public Security Secretariat (SSP).  
🔹 Responsible for **designing and coding the UI** using **Vue.js** and **Bootstrap**, creating an intuitive and dynamic user experience.  
🔹 Implemented a **Content Management System (CMS)** to manage news, informational articles, and interactive videos.  

---

### 🏤 **Web System for the DIF Municipal Xalapa Support Management**  
📅 **July 2024 - October 2024**  
🔹 Developed the **frontend** of the DIF Municipal Xalapa support management system using **React.js and Bootstrap**.  
🔹 Integrated various modules with the **API** and implemented UI components.  
🔹 Worked with **MySQL** for data management, utilizing **Amazon RDS** for the database and **Amazon S3** for file storage.  
🔹 The system was deployed on an **AWS EC2 instance** and allows the administration of requests, neighborhoods, inventories, and beneficiary records.  

---

### 🌿 **PWA for Forest Traceability Management of Plants and Seeds**  
📅 **August 2024 - January 2025**  
🔹 Developed a **Progressive Web App (PWA)** for traceability management at the **Red de Viveros de Biodiversidad (REVIVE)** organization.  
🔹 Designed and implemented the UI using **React.js with TailwindCSS**, visually representing processes from seed collection to distribution.  
🔹 Configured the **Service Worker** to enable offline functionality, ensuring access to information without an internet connection.
